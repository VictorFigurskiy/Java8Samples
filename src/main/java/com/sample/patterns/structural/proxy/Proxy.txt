Шаблон Прокси (Proxy)
==========================================================
Цель:

Для замещения другого объекта и контроля доступа к нему.

==========================================================

Используется для обеспечения контроля доступа к определенному объекту.

==========================================================
Пример использования:

- когда есть необходимость ссылаться на объект способом отличным от обычного указателя;

    * удаленное замещение -     локальный представитель вместо объекта
                                в другом адресном протранстве;

    * виртуальное замещение -   создание "тяжелых объектов" при необходимости;

    * защищенное замещение -    контроль доступа к указанному объекту;

    * "умная" ссылка -          замена обычного указателя
                                (подсчет числа ссылок, установка блокировки и т.д).

==========================================================
